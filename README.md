# Fashion Market Pricing Comparison & Analytics Pipeline

This project compares pricing of Ariseism and its Western competitor Brandy Melville to understand market positioning and feasibility for expansion into North America.
It's an end-to-end data pipeline that collects, cleans, and analyzes retail pricing data from multiple e-commerce platforms. The goal is to transform raw, unstructured web data into a validated relational dataset and perform statistical analysis to identify pricing differences across markets.

Features
	•	Scrapes 300+ product listings from multiple e-commerce sources
	•	Transforms raw HTML data into a normalized relational schema
	•	Applies SQL-based joins, filtering, and validation logic
	•	Performs statistical analysis to identify cross-market pricing discrepancies
	•	Modular pipeline design separating ingestion, storage, and analysis layers

Tech Stack
	•	Python – core data processing and analysis
	•	Pandas – data transformation and statistical analysis
	•	SQL – relational storage, joins, filtering, and validation
	•	BeautifulSoup – web scraping and HTML parsing
    Seaborn - data visualization
  
Outcome: Visualizations (boxplots, histograms) showing price distributions by brand and product type, insights on affordability, and target price ranges of clothing products.
